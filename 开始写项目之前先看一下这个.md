# AIDE Pro 编译约束提示词

> 将以下内容粘贴给 AI，确保生成的代码在 AIDE Pro 上零报错编译通过。

---

## 核心约束

所有代码必须满足：**Gradle 4.x + ECJ 编译器 + 纯 Android SDK 26 + 单模块 + 零远程依赖**

---

## 1. Gradle 配置名

- ✅ 依赖用 `compile`，**禁止**用 `implementation`、`api`、`compileOnly`、`runtimeOnly`
- ✅ `compileSdkVersion` 设为 `26`
- ✅ `targetSdkVersion` 设为 `24`
- ✅ `buildToolsVersion` 设为 `"26.0.0"`
- ✅ `minSdkVersion` 任意，建议 `19`
- ✅ proguard 用 `proguard-android.txt`，**禁止**用 `proguard-android-optimize.txt`

### 正确的 build.gradle 模板

```groovy
apply plugin: 'com.android.application'

android {
    compileSdkVersion 26
    buildToolsVersion "26.0.0"

    defaultConfig {
        applicationId "com.example.app"
        minSdkVersion 19
        targetSdkVersion 24
        versionCode 1
        versionName "1.0"
    }

    buildTypes {
        release {
            minifyEnabled false
            proguardFiles getDefaultProguardFile('proguard-android.txt'), 'proguard-rules.pro'
        }
    }
}

dependencies {
    compile fileTree(dir: "libs", include: ["*.jar"])
}
```

---

## 2. Java 语法限制（ECJ 编译器）

AIDE 使用 Eclipse Compiler for Java，以下语法**全部禁止**：

| 禁止 | 替代写法 |
|------|----------|
| Lambda `(a, b) -> { }` | 匿名内部类 |
| 方法引用 `Class::method` | 匿名内部类 |
| Stream API `list.stream()` | for 循环 |
| try-with-resources 多变量 | try-finally 手动 close |
| `var` 关键字 | 显式类型声明 |
| `switch` 表达式（Java 14） | 传统 switch |
| text block `"""..."""` | 普通字符串拼接 |

### Lambda 转匿名内部类示例

```java
// ❌ 禁止
list.sort((a, b) -> a.compareTo(b));

// ✅ 正确
Collections.sort(list, new Comparator<String>() {
    @Override
    public int compare(String a, String b) {
        return a.compareTo(b);
    }
});
```

---

## 3. 包导入限制

以下包在 AIDE 默认 SDK 中**不存在**，禁止 import：

- `androidx.*`（整个 androidx 包族）
- `com.google.android.material.*`
- 任何第三方 Maven 依赖（除非已将 jar 放入 `libs/`）

| 禁止 | 替代 |
|------|------|
| `androidx.core.content.FileProvider` | `Uri.fromFile()` |
| `androidx.appcompat.app.AppCompatActivity` | `android.app.Activity` |
| `androidx.recyclerview.widget.RecyclerView` | `ListView` |

允许的包：`android.*`、`java.*`、`javax.*`、`org.w3c.*`、`org.xml.*`

---

## 4. 远程依赖

- **禁止**在 `dependencies` 中添加任何远程 Maven 坐标
- AIDE 的 Maven 拉取经常超时，且**静默失败**——不报网络错误，只报 `unknown type or package`
- 如需第三方库，下载 jar 放到 `app/libs/`，用 `compile fileTree` 引入

---

## 5. 项目结构

- **单模块**项目。AIDE 对多模块 (`:library`) 支持很差，跨模块资源引用经常失败
- 布局文件放在 `res/layout/`，资源名全小写字母+下划线

---

## 6. 清单文件注意事项

- `AndroidManifest.xml` 中**不要**声明 `FileProvider`（需要 `androidx`）
- `resizeableActivity` 可选
- 权限声明只用标准 Android 权限和 `moe.shizuku.manager.permission.API_V23`

---

## 7. 布局文件注意事项

- `activity_main.xml` 放 `res/layout/`
- **不要**在 XML 布局中使用 `androidx` 控件（如 `androidx.appcompat.widget.SwitchCompat`）
- 使用原生控件：`Button`、`TextView`、`Switch`、`ScrollView`、`LinearLayout`
- `Switch` 用 `android.widget.Switch`，不是 `SwitchCompat`

---

## 8. 文件编码

- 全部 UTF-8
- 禁止半角/全角混用引号（`"` vs `"`），ECJ 会报 `wrong "`
- 粘贴外部代码后使用 AIDE 的 Format Code 功能修正

---

## 9. 常见报错速查

| 现象 | 真凶 |
|------|------|
| `unknown type or package` | 远程依赖未下载 / import 了 `androidx` |
| `Unexpected end of declaration` | Lambda / 方法引用 / try-with-resources |
| `unknown variable or field` | Lambda 参数解析失败 |
| `wrong "` / `missing )` | 全角引号混入 |
| `Failed to resolve` | Maven 依赖拉取超时 |

---

## 对话时直接告诉 AI

> 用 AIDE Pro 编译，要求：Gradle 4.x，compileSdkVersion 26，Java 6/7 语法，零远程依赖，纯 android.* / java.* 包，不用 lambda，不用 androidx，单模块项目。
